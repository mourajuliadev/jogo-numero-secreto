# Jogo do Número Secreto

Projeto desenvolvido utilizando HTML, CSS e JavaScript com foco na prática de lógica de programação e manipulação do DOM.

## Funcionalidades

- Geração aleatória de número secreto
- Validação de tentativas
- Dicas de número maior ou menor
- Contador de tentativas
- Reinício do jogo
- Controle para evitar repetição de números sorteados
- Feedback visual e sonoro ao usuário

## Conceitos praticados

- Manipulação do DOM
- Funções em JavaScript
- Condicionais
- Arrays
- Recursão
- Template strings
- Eventos
- Atualização dinâmica da interface

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo

Projeto criado para praticar conceitos fundamentais de desenvolvimento web e lógica de programação utilizando JavaScript.
